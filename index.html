Contractors Access Levels and Risk Management

This dashboard lists all contractors and third-party employees for a fake company. It analyzes permissions, production access, and activity patterns to generate security recommendations. It's built with AWS Lambda and AWS RDS (PostgreSQL) for serverless backend processing, Google Gemini for LLM risk analysis, and JavaScript for the frontend design.
